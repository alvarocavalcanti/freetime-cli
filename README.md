# freetime-cli

> CLI tool for register new posts on free-time

## Installation

Install with NPM:

```sh
[sudo] npm i -g freetime-cli
```

## Usage

```sh
freetime
```

## Questions

#### Title 
Post title

#### Duration:
Talk duration

#### Categories:
Categories (separated by commas)

#### Image Path
Image Path (absolute or relative)

#### URL
Talk link

#### Path to save
Path to save file (absolute or relative)
> _default_: actual directory
